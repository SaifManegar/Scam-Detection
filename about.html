Project Overview

The Scam Detection System is a machine learning-based application designed to identify spam or scam messages accurately. It utilizes natural language processing techniques and various classification algorithms to distinguish between genuine and fraudulent messages.

Preprocessing Steps

Before training the machine learning models, the dataset undergoes several preprocessing steps to enhance the quality of the input data:

  • Conversion of text to lowercase
  • Removal of special characters and digits
  • Elimination of links
  • Tokenization of text
  • Exclusion of stopwords
  • Stemming of words

Models Training

The Scam Detection System employs three machine learning models:

  • Random Forest Classifier
  • Logistic Regression Classifier
  • Support Vector Machine (SVM)

Each model is trained on preprocessed data using TF-IDF vectorization to extract features and make predictions.

Performance Metrics

Model Accuracy Precision Recall F1-score
Random Forest 0.98 0.99 0.92 0.95
Logistic Regression 0.96 0.97 0.88 0.92
SVM 0.98 0.98 0.95 0.96

Detection System

The detection system is the core component of the Scam Detection System. It utilizes the trained machine learning models to analyze input text and determine whether it is genuine or fraudulent.

The function predict_fake_or_real() cleans the input text, transforms it using TF-IDF vectorization, and predicts the authenticity of the message based on the trained Random Forest classifier.

Integration into Website

The Scam Detection System can be seamlessly integrated into a website to provide real-time scam detection services. Website visitors can input text messages, and the system will promptly classify them as genuine or fraudulent.

Example usage:


            input_text = "I'm gonna be home soon and i don't want to talk about this stuff anymore tonight, k? I've cried enough today.\n"
            prediction = predict_fake_or_real(input_text)
            print("Prediction:", prediction)
